An optimized, premium **RAG (Retrieval-Augmented Generation)** document intelligence platform. KnowDoc dynamically extracts text from mixed selectable/scanned PDFs and images, scales visual layout coordinates to standard grids for visual box overlays, and serves an interactive conversational workspace with persistent session-synced chats.

---

## 🛠️ Main Technology Stack

*   **Frontend**: Next.js 16.x (App Router), React 19.x, ReactMarkdown (with GFM), Vanilla CSS Modules.
*   **Backend**: Python FastAPI, CPU-bound PaddleOCR, PyMuPDF (`fitz`), ChromaDB (vector storage), SentenceTransformers, Google Gemini 2.5 Flash.
*   **Database & Stores**: Local persistent JSON databases (`documents.json`, `chats.json`, `folders.json`) protected by thread-safe locking structures.

## 🚀 Full Setup & Setup Commands

To configure and run the entire KnowDoc stack locally on your computer, please follow these detailed steps:

### 1. 🦾 Backend System Setup & Start
The backend parses documents, computes semantic embeddings, indexes vectors, and queries the generative Google Gemini RAG model.

#### Setup Prerequisites:
Make sure you have your virtual environment configured and active from the root folder:
```bash
# 1. Activate the Virtual Environment (Windows PowerShell)
venv\Scripts\activate
```

#### Dependencies Installation:
```bash
# 2. Navigate to backend directory
cd backend

# 3. Install required libraries
pip install -r requirements.txt
```

#### Configure Environment Variables:
Create a `.env` file inside the `backend/` directory and configure your Google Gemini API key:
```env
GEMINI_API_KEY=AIzaSy...your_actual_gemini_key
```

#### Run uvicorn API Server:
Start the server mapping localhost port `8000`:
```bash
# 4. Start the FastAPI development server
..\venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```
*   **Local endpoints url**: `http://127.0.0.1:8000/`
*   **Interactive Swagger Documentation**: `http://127.0.0.1:8000/docs`

---

### 2. 💻 Frontend Client Setup & Start
The frontend presents a gorgeous 3-pane auditory inspect dashboard, handles drag-and-drop ingestion, and streams RAG conversations.

#### Dependencies Installation:
Open a second PowerShell window, navigate to the frontend directory, and install requirements:
```bash
# 1. Navigate to frontend directory
cd frontend

# 2. Install all node packages
npm install
```

#### Start Next.js Development Server:
Launch the dev server with high-performance Turbopack compiler enabled:
```bash
# 3. Start Next.js client server
npm run dev
```
*   **Workspace interface url**: Open [http://localhost:3000/workspace](http://localhost:3000/workspace) in your web browser.
